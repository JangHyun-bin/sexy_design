# Dark Technical Editorial

Source signal: Reference A.

## Use When

Use for automation, AI operations, security, developer tools, infrastructure,
monitoring, and products that should feel precise, dark, and operational.

## Surface

- black or near-black paper
- subtle halftone, dotted grid, or hairline frame
- low-contrast panels
- small hot accent, usually orange or red-orange
- dashboard area can be the main object

## Typography

- display: white editorial serif or sharp high-contrast serif
- body: neutral sans in muted gray
- labels: mono or uppercase grotesque
- numbers: tabular, high contrast

## Dominant Gesture

One of:

- dark dashboard surface
- command-center hero
- large serif statement over operational proof
- chart or task system

## Interaction Stance

- still and precise
- small state signals
- chart reveals
- no bouncy motion
- no playful hover scale

## Good Components

- operational dashboard surface
- task table
- status strip
- logo wall in hairline cells
- mono labels
- chart block
- outline buttons

## Avoid

- blue SaaS gradient
- friendly mascot
- colorful icon cards
- excessive glow
- fake terminal without real command meaning
