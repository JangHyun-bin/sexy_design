# Industrial Swiss Object

Source signal: References E and F.

## Use When

Use for hardware, watches, manufacturing, tools, sensors, automotive, luxury
objects, industrial brands, and any product where the object can carry authority.

## Surface

- white, gray, black
- industrial neutral field
- hard product shadows
- almost no decoration
- huge negative space
- annotations placed like engineering labels

## Typography

- display: giant grotesque or engineered wordmark
- body: compact sans
- labels: mono technical annotation
- numbers/specs: tabular, sparse

## Dominant Gesture

One of:

- huge wordmark
- product object hero
- technical annotation field
- monochrome catalog surface

## Interaction Stance

- still and confident
- product rotation or scroll reveal only if implemented well
- precise hover, no bounce
- mechanical state changes

## Good Components

- giant wordmark hero
- object-first hero
- tabular spec sheet
- technical annotation
- minimal nav
- industrial product grid

## Avoid

- rounded SaaS cards
- colorful icons
- friendly copy
- multiple CTAs in the hero
- fake 3D if the product render is weak
