# Giant Wordmark Hero

## Use When

Use for industrial, fashion, product, studio, and brand-first pages where the
wordmark itself is the object.

## Anatomy

- edge-aligned or minimal nav
- oversized wordmark
- small technical or editorial annotations
- product object or proof below
- sparse CTA

## Layout Rules

- Wordmark can crop or nearly touch edges if intentional.
- Annotation copy should be small and precise.
- Keep layout quiet around the wordmark.
- Do not add decorative icons to make it friendlier.

## Typography Rules

- Wordmark is the display type.
- Supporting text can be mono, compact sans, or small serif.
- Avoid multiple display families.

## Surface Rules

- Black, white, gray, or one strong field color works best.
- Texture should be minimal unless it supports brand physicality.

## Responsive Behavior

- Mobile can show cropped wordmark, stacked letters, or abbreviated monogram.
- Maintain confidence; do not shrink the mark into a normal logo.

## Anti-Slop Checks

- Fails if the wordmark becomes ordinary nav branding.
- Fails if the hero adds a generic headline that repeats the wordmark.
- Fails if CTAs visually overpower the brand gesture.
