# Mobile Kinetic Finance Prototype

This is the first dogfood output from Design Operating Stack V1.

## Stack Choices

- Mode: `generate`
- Genre: `mobile-kinetic-finance`
- Mobile patterns: `card-stack-wallet`, `transaction-dashboard`, `bottom-action-system`
- Dominant gesture: oversized balance plus selected debit card
- Accent footprint: signal/field through red currency markers and blue card depth
- Surface: dark finance depth, dot matrix, frosted white transaction sheet

## Files

- `index.html`: static prototype
- `styles.css`: visual system and responsive states
- `preview-390.png`: Chrome headless render at 390 px wide

## View

Open `index.html` directly in a browser. No build step is required.
