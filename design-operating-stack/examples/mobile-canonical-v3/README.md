# Mobile Canonical V3

Canonical direction after the V2 mutation board.

## Direction

V3 merges the strongest V2 ingredients into one agent-facing mobile operating
screen:

- F1/F2: active run console and operational queue
- A1/A2: mobile object authority, selected state, and bottom action sheet
- C1/C2: industrial spec language and control-state material

## Files

- `index.html`: single-file HTML/CSS/JS canonical design
- `preview-1440.png`: desktop capture
- `preview-500.png`: narrow viewport capture
