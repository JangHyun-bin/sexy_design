# Mobile Shotgun Board

One-page comparison board for six mobile UI directions generated from Design
Operating Stack V1.

## Variants

- A. Night Wallet: mobile kinetic finance, oversized balance, card stack
- B. Acid Alarm: time picker, acid yellow flood, tactile ruler
- C. Swiss Vault: industrial object-first wallet
- D. Notebook Pay: academic paper and AI assistant grammar
- E. Care Ledger: warm institutional trust direction
- F. Ops Console: dark technical operational dashboard

## Files

- `index.html`: single-file HTML/CSS/JS comparison board
- `scores.md`: scoring notes using the taste rubric
- `preview-1440.png`: full desktop board capture
- `preview-500.png`: narrow viewport capture

## Local Server

From the repository root:

```powershell
python -m http.server 4174 --bind 127.0.0.1 --directory .\design-operating-stack\examples\mobile-shotgun-board
```

Then open `http://127.0.0.1:4174/`.
